# KAF-305 (kaf305)

Single-page application for the 305 department of the Moscow Aviation Institute. It is built on the basis of the Quasar Framework. The architecture is based on the composition-api.

## Install the dependencies
```bash
yarn
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
quasar dev
```

### Lint the files
```bash
yarn run lint
```

### Build the app for production
```bash
quasar build
```

### Customize the configuration
See [Configuring quasar.conf.js](https://v2.quasar.dev/quasar-cli/quasar-conf-js).
